# Changelog — SoberFurry MultiNecklaces

## 1.0.0
Первый выпуск.
- Несколько разных ожерелий на одном последователе; запрет дубликатов одного типа.
- Управление в круговом меню последователя: «Ожерелья» (иконка = надетое) → Показать / Скрыть / Снять,
  закреплённый «Подарить», глаз/зачёркнутый глаз, авто-листание. Запасная панель — F8.
- Все надетые ожерелья видны в «Прочитать мысли».
- Эффекты скрытых ожерелий складываются (скорость, сбор, преданность, демон, бессмертие, жизнь ×2,
  сон, лояльность, мороз/зима/гниение и др.). Death's Door остаётся одноразовым.
- Лояльность при надевании/снятии, как в оригинале.
- Своё хранилище на слот (атомарная запись, бэкап), смерть/воскрешение, выпадение всех при разделке
  (без дубликатов).
- Опциональная интеграция с COTL_API (вкладка «Mods»), русская и английская локализация.
