# Sam MultiNecklaces

BepInEx-мод для **Cult of the Lamb**: один последователь может носить **несколько ожерелий** сразу.
Эффекты всех надетых действуют, на модели видно одно выбранное, остальные работают скрытыми.

**Версия:** 1.5.0 · **Зависимости:** BepInEx 5.4.x, COTL_API 0.3.4 (необязательно)

## Установка
Скачай из [Releases](../../releases) и распакуй в:
```
<Папка игры>\BepInEx\plugins\SamMods\MultiNecklaces\
```

## Как пользоваться
Открой круговое меню последователя → пункт **«Ожерелья»** (иконка = надетое ожерелье):
- **«Подарить ожерелье»** — надеть ещё одно;
- клик по ожерелью → **Сделать видимым / Скрыть / Снять**;
- глаз = на модели, зачёркнутый глаз = скрыто; много ожерелий — листается само.

Все надетые ожерелья видны в экране **«Прочитать мысли»**. Запасной интерфейс — клавиша **F8**.

## Эффекты
Эффект видимого ожерелья работает всегда. У скрытых сейчас складываются: **преданность** (Necklace_1),
**сбор ресурсов** (Necklace_4), **не замерзает** (Frozen), **+2 демона** (Demonic), **бессмертие**
(Gold_Skull). Остальные действуют только когда ожерелье видимое (добавлю в следующих версиях).

## Сборка из исходников
```
dotnet build src/Sam.MultiNecklaces/Sam.MultiNecklaces.csproj -c Release -p:GamePath="X:\путь\к\Cult of the Lamb"
```
