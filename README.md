## ✨ Возможности

- 👑 **Несколько разных ожерелий** на одном последователе одновременно
- 👁️ Одно **видимое** на модели — остальные действуют **скрытыми**
- 🎡 Управление прямо в **круговом меню** последователя (удобно с геймпада)
- 🎁 **Подарить** ещё ожерелье, не выходя из меню
- 📜 Все надетые ожерелья видны в экране **«Прочитать мысли»**
- 💾 Своё сохранение на каждый слот; безопасно при смерти, воскрешении и разделке

## 📦 Установка

1. Установите **BepInEx** (пак для Cult of the Lamb).
2. Скачайте архив из **[Releases](../../releases)** и распакуйте в:

```
<Папка игры>\BepInEx\plugins\SoberFurryMods\MultiNecklaces\
```

## 🎮 Как пользоваться

Откройте круговое меню последователя → пункт **«Ожерелья»** *(иконка = надетое ожерелье)*:

| Действие | Что делает |
|---|---|
| 🎁 **Подарить ожерелье** | надеть ещё одно из инвентаря |
| 👁️ **Сделать видимым** | показать это ожерелье на модели |
| 🚫 **Скрыть** | убрать с модели (эффект остаётся) |
| ❌ **Снять** | вернуть ожерелье в инвентарь |

Глаз = на модели · зачёркнутый глаз = скрыто · при большом числе ожерелий меню **листается само**.

## ⚡ Эффекты

С **v1.1.0** эффекты **скрытых** ожерелий тоже складываются: скорость, сбор, преданность, демон,
бессмертие, продолжительность жизни ×2, сон, лояльность, защита от мороза/зимы/гниения, спасение от
смерти и др. Отключается в конфиге `Effects/EffectStacking = false`.

📖 **Полный справочник всех 17 ожерелий — [docs/NECKLACES.md](docs/NECKLACES.md)**
(что делает каждое, что стакается, поведение при смерти/разделке/снятии).
