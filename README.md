# Sam MultiNecklaces

BepInEx-плагин для **Cult of the Lamb**: один последователь может носить **несколько разных
ожерелий** одновременно. Эффекты всех надетых ожерелий действуют; на модели показывается одно
выбранное («видимое»), остальные продолжают работать скрытыми.

- Версия: **1.5.0**
- GUID: `com.sam.cultofthelamb.multinecklaces`
- Собрано под: Unity 2022.3.62f2, build 22885603
- Зависимости: BepInEx 5.4.x (пак для COTL), COTL_API 0.3.4 (мягкая)

## Установка (для игроков)
Скопировать в:
```
<Папка игры>\BepInEx\plugins\SamMods\MultiNecklaces\
```
Конфиг создаётся при первом запуске: `BepInEx\config\com.sam.cultofthelamb.multinecklaces.cfg`.

## Как пользоваться — через круговое меню последователя
Управление встроено в **круговое меню взаимодействия** (то же, где «Дать работу» и т.п.):

1. Подойди к последователю, открой круговое меню.
2. Выбери пункт **«Ожерелья»** — его иконка = текущее надетое (видимое) ожерелье.
3. Откроется под-меню:
   - сверху закреплён пункт **«Подарить ожерелье»** — надеть ещё одно из инвентаря;
   - дальше — по сектору на каждое надетое ожерелье. На каждом — **глаз** (отображается на персонаже)
     или **зачёркнутый глаз** (скрыто).
4. Нажми на ожерелье → откроется его меню действий:
   - **Сделать видимым** (если скрыто) или **Скрыть** (если отображается),
   - **Снять** (возвращает предмет).
5. Если ожерелий много — меню **листается автоматически** (стрелки, как в ванильном колесе).

Наведение на любой пункт показывает **название и эффект** ожерелья.

> Запасной интерфейс: панель по клавише **F8** (настраивается, `UI/PanelHotkey`) — на случай, если
> круговое меню недоступно. Основной способ — круговое меню.

## Где видно надетые ожерелья
В экране **«Прочитать мысли»** показываются **все** надетые ожерелья (и видимые, и скрытые) с
иконками, названиями и описаниями.

## Лояльность
Как в оригинале: надевание ожерелья даёт последователю обожание/лояльность (+ положительную мысль),
снятие — добавляет отрицательную мысль.

## Эффекты скрытых ожерелий
Эффекты **видимого** ожерелья работают всегда (ванильно). Для **скрытых** на данный момент
складываются те эффекты, что в игре заданы «чисто» и патчатся безопасно:

| Ожерелье | Эффект | Стакается скрытым |
|---|---|---|
| Necklace_1 | +преданность | ✅ |
| Necklace_4 (природы) | +сбор ресурсов | ✅ |
| Necklace_Frozen | не замерзает | ✅ |
| Necklace_Demonic | +2 уровня демона | ✅ |
| Necklace_Gold_Skull | бессмертие | ✅ |
| прочие (скорость, без сна, спасение от смерти, лояльность и т.п.) | разное | ⚠️ только когда видимое |

Остальные эффекты вшиты в большие методы игры и будут добавляться по одному в следующих версиях.

## Сохранение и безопасность
- Своё JSON-хранилище **на каждый слот** (`saves\Sam.MultiNecklaces.slot{N}.json`): атомарная запись,
  версия схемы, резервная копия `.bak`, копия перед миграцией.
- Ключ последователя = постоянный `FollowerInfo.ID`.
- Смерть не снимает ожерелья; воскрешение сохраняет набор.
- **Разделка трупа**: все оставшиеся ожерелья выпадают по одному (повторный вызов не дублирует).
- Перед удалением мода — вернуть все скрытые ожерелья в инвентарь (кнопка в F8-панели).

## Сборка из исходников
Нужен .NET SDK 8 и установленная игра с BepInEx + COTL_API.
```
dotnet build src/Sam.MultiNecklaces/Sam.MultiNecklaces.csproj -c Release -p:GamePath="X:\путь\к\Cult of the Lamb"
```
Путь к игре также можно задать переменной окружения `COTL_GAMEPATH`. Игровые DLL подключаются по
`HintPath` и **не** входят в репозиторий.

## Удаление
Удалить папку `BepInEx\plugins\SamMods\MultiNecklaces\` и файлы `saves\Sam.MultiNecklaces.slot*.json`.
