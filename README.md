## ✨ Возможности

- 👑 **Несколько разных ожерелий** на одном последователе одновременно
- 👁️ Одно ожерелье **видимо** на модели — все остальные действуют **скрытыми**
- ⚡ Эффекты **всех** надетых ожерелий **складываются**, даже у скрытых
- 🎡 Управление прямо в **круговом меню** последователя — удобно с геймпада
- 🎁 **Подарить** ещё ожерелье, не выходя из меню
- 📜 Все надетые ожерелья видны в экране **«Прочитать мысли»**
- 💾 Своё сохранение на каждый слот; безопасно при смерти, воскрешении и разделке
- 🧩 Без дубликатов: один тип — один раз; одноразовые остаются одноразовыми

## 📦 Установка

1. Установите **[BepInEx](https://thunderstore.io/c/cult-of-the-lamb/p/BepInEx/BepInExPack_CultOfTheLamb/)** для Cult of the Lamb.
2. Скачайте архив из **[Releases](../../releases/latest)** и распакуйте в:

```
<Папка игры>\BepInEx\plugins\SoberFurryMods\MultiNecklaces\
```

---

## 🎮 Как пользоваться

Откройте круговое меню последователя → пункт **«Ожерелья»** *(его иконка = надетое ожерелье)*:

| Действие | Что делает |
|---|---|
| 🎁 **Подарить ожерелье** | надеть ещё одно из инвентаря |
| 👁️ **Сделать видимым** | показать это ожерелье на модели |
| 🚫 **Скрыть** | убрать с модели *(эффект остаётся)* |
| ❌ **Снять** | вернуть ожерелье в инвентарь |

> 👁️ глаз — на модели · 🚫 зачёркнутый глаз — скрыто · при большом числе ожерелий меню **листается само**.

## ⚡ Эффекты

Эффекты **скрытых** ожерелий тоже складываются: скорость, сбор ресурсов, преданность, уровень демона,
бессмертие, продолжительность жизни ×2, гарантированный сон, лояльность, защита от мороза / зимы /
гниения, спасение от смерти и другие.

> 📖 **Полный справочник всех ожерелий — [docs/NECKLACES.md](docs/NECKLACES.md)**
> *(что делает каждое, что складывается, поведение при смерти / разделке / снятии).*

## ⚙️ Настройки

Правятся в файле `BepInEx\config\com.soberfurry.cultofthelamb.multinecklaces.cfg`.
Если установлен **COTL_API** *(необязательно)* — те же настройки появятся во вкладке **«Mods»** прямо
в настройках игры, на русском языке.
