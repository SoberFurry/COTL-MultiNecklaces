# 📖 Справочник ожерелий

Мод **не добавляет** новые ожерелья — он работает со всеми **17 ванильными** ожерельями игры и
позволяет носить их вместе. Начиная с **v1.1.0** эффекты **скрытых** ожерелий тоже складываются.

Обозначения:
- ✅ **Стак** — эффект работает даже когда ожерелье **скрыто**.
- 🎀 **Косметика/неясно** — явного игрового эффекта в коде не нашёл.

## Полный список

|---|---|---|

| `Necklace_2` | +25% к скорости передвижения («с перьями») | ✅ |
| `Necklace_3` | продолжительность жизни ×2 («с черепом») | ✅ |
| `Necklace_4` | +25% к сбору ресурсов («природы») | ✅ |
| `Necklace_5` | влияет на сон (взаимодействие с «Бессонницей») | ✅ |
| `Necklace_Loyalty` | не становится несогласным (лояльность) | ✅ |
| `Necklace_Demonic` | +2 к уровню демона | ✅ |
| `Necklace_Gold_Skull` | бессмертие | ✅ |
| `Necklace_Deaths_Door` | разовое спасение от смерти | ✅ *(одноразовое)* |
| `Necklace_Winter` | защита зимой + бонус продуктивности зимой | ✅ |

## Как это работает
Все эффекты выше читаются игрой через проверки «надето ли ожерелье X». Мод заменяет эти проверки так,
что они учитывают **все надетые** ожерелья (видимое + скрытые), а не только видимое. Поэтому скрытое
ожерелье продолжает давать эффект. Видимое ожерелье по-прежнему отображается на модели.

Отключить стак можно в конфиге: `Effects/EffectStacking = false` (тогда поведение как в ванили —
эффект только у видимого).

## Поведение при событиях
- **Смерть:** ожерелья не теряются — сохраняются за последователем.
- **Воскрешение:** весь набор восстанавливается.
- **Разделка трупа:** все оставшиеся ожерелья выпадают по одному (без дублей).
- **Снятие:** любое ожерелье снимается через круговое меню; неснимаемых среди ванильных нет.

> `Necklace_Dark / Light / Bell / Missionary` помечены «🎀» честно — явного эффекта в коде игры я не
> нашёл (возможно, только внешний вид или эффект завязан на отдельную механику).
